据称，所谓 PS5 Pro 的 GPU 将具有 300 TOPS 的机器学习算力（8bit 整数计算），以及 67 TFLOPS 的 16bit 浮点算力，FP32 单精度算力估计为 33.5 TFLOPS，是当前 PS5 的 3 倍以上。如此高的 GPU 算力表明其频率也会很高（2.18~2.45 GHz），之前的爆料显示 PS5 Pro 会采用定制 APU，其中 GPU 将启用 60 个 CU（总共 64 个）。在 MLID 放出的这份“Trinity 技术概述”文档中显示（“Trinity” 是 PS5 迭代机型的代号，自 2022 年初以来一直在开发中），支持机器增强版 TAAU (Temporal Anti Aliasing Upscale) 技术。这一技术类似于 DLSS 或 FSR，具有完整的 HDR 支持。该技术还将支持动态分辨率，并在未来更新支持最高 8K 分辨率输出。从 1080p 超分至 4K 分辨率需要 2ms 的 GPU 周期时间，这一方面仍需要进一步优化，而且这样的库只需要 250MB 的内存。与 AMD FSR 和 NVIDIA DLSS 一样，这一技术不需要针对每款游戏进行特意适配，这意味着开发人员只需通过官方软件开发工具包即可轻松地支持 PSSR。[url=]<img src="https://assets.msn.cn/staticsb/statics/latest/views/icons/ArticleImageFullscreen.svg" id="aimg_FX4S7" lazyloadthumb="1" onclick="zoom(this, this.src, 0, 0, 0)" onmouseover="img_onmouseoverfunc(this)"/)[/url]
